Transfer Syntax Last Updated 11/01/2013

A transfer syntax is a set of encoding rules that allow Application Entities to unambiguously negotiate the encoding techniques (e.g., Data Element structure, byte ordering, compression) they are able to support, thereby allowing these Application Entities to communicate.

DICOM Standard 3.10
The Transfer Syntax UID is the unique identifier used as a tag in the DICOM standard to determine the type of encoding principles used for any given DICOM file. The Transfer Syntax UID is in the file DICOM Tag field (0002,0010). The transfer syntaxes highlighted in green are currently supported by dcmdb, the ones in red are not supported.
